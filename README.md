# docker-geant4
This build will contain Geant4 data.  So the final image size is >7GB.

## Versions
- Geant4 10.5.0

From Docker image mazurov/cern-root
- CentOS 7.3
- Root 6.09.02
- CMake 3.6.3

Build and run scripts are included to ensure reproducibility.

- docker-run.sh
